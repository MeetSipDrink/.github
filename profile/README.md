# **한 마디 한 잔**

## **팀원**
- 남채연
- 방승욱
- 김찬준
- 서진형

## **유저 플로우 & ERD**
- [유저 플로우](https://drive.google.com/file/d/1JRvUMDukagLtbae6EEjZQ6sDfLmo4_2r/view)

## **기술 스택**
- **Frontend:** React Native
- **Backend:** Spring Boot
- **Database:** MySQL
- **WebRTC:** Peer-to-Peer Communication
- **WebSocket:** 실시간 채팅
